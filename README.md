# Drawing App
## Description
A simple and lightweight Tkinter drawing application built in Python.
This app allows users to draw freehand lines on a canvas, change brush color, and adjust brush thickness using a slider.
It is designed as a beginner‑friendly project demonstrating event‑driven programming, GUI layout, and basic drawing logic.

## Features
- Freehand drawing
- Adjustable brush thickness (1–20 px)
- Color picker for custom brush colors
- Clear button to reset the canvas

## How to Run
### Requirements
- Python 3.x
- Tkinter
### Run the app
python drawing_app.py
